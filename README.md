# Sri Lanka Agricultural Insights

This repository provides an **exploratory data analysis (EDA) and business insights** on agricultural production and export data in Sri Lanka. The project includes data cleaning, visualization, trend analysis, and a **Streamlit dashboard** for interactive exploration.

---

## Project Overview

The main goals of this project are to:

- Explore agricultural production and export data
- Perform data cleaning
- Conduct exploratory data analysis (EDA)
- Generate business insights
- Provide an interactive dashboard using Streamlit

---

## Project Structure

The project is organized as follows:

- **app/**: Contains the Streamlit app and related files.
- **data/**: Contains the raw data files.
- **notebooks/**: Contains Jupyter notebooks for data cleaning, EDA, and business insights.
- **reports/**: Contains the generated reports and figures.

---

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository.
2. Create a virtual environment.
3. Install the dependencies.
4. Run the Streamlit app.

---

## Data Sources

The data used in this project is obtained from official Sri Lankan agricultural statistics sources. The data files used are:

- `production_of_major_agricultural_crops.csv`: National crop production data.
- `volume_and_value_of_export_agriculture_crops.csv`: Export volumes and values.
